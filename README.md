# 🎓 Learning Management System

A modern and scalable Full Stack Learning Management System (LMS) built using React.js, Spring Boot, and PostgreSQL. This platform is designed to provide an interactive and efficient digital learning environment for students and instructors.

The project demonstrates complete full stack web development by integrating frontend, backend, REST APIs, authentication, and database management into a real-world educational platform.

---

# 🚀 Project Overview

The Learning Management System is a web-based platform where users can register, log in, access learning resources, and manage educational activities through a clean and responsive interface.

This application focuses on delivering a smooth user experience with secure authentication, structured course management, and scalable backend architecture.

The system is designed using modern technologies and follows industry-level development practices.

# ⚙️ How the System Works

1. Users create an account using the registration system.
2. Registered users can securely log in.
3. Students can access courses and educational resources.
4. Frontend communicates with backend APIs.
5. Spring Boot handles server-side logic and requests.
6. PostgreSQL stores and manages application data securely.

# 💻 Installation & Setup

## Clone Repository

```bash
git clone https://github.com/vishal2580k/learning-management-system.git
```

---

# ▶️ Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

Frontend will run on:

```bash
http://localhost:5173
```

---

# ▶️ Backend Setup

```bash
cd backend
mvn spring-boot:run
```

Backend server will run on:

```bash
http://localhost:8080
```

---

# 🗄️ Database Configuration

1. Install PostgreSQL
2. Create a new database
3. Configure database credentials inside `application.properties`

Example:

```properties
spring.datasource.url=jdbc:postgresql://localhost:5432/lms
spring.datasource.username=postgres
spring.datasource.password=yourpassword
```
